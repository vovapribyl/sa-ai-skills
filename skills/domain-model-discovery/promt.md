Ты выступаешь в роли ведущего системного аналитика.

На основе предоставленного описания предметной области выполни выделение доменной модели.

Выполни следующие шаги:

1. Выдели бизнес-сущности.
2. Для каждой сущности определи основные атрибуты.
3. Выяви связи между сущностями.
4. Определи кратности связей.
5. Определи кандидатов в агрегаты DDD.

# Атрибуты сущностей

# Связи

# Кандидаты в агрегаты
