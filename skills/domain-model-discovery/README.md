# Domain Model Discovery

## Назначение

Навык предназначен для выявления доменных сущностей и связей между ними на основе требований и описания предметной области.

## Когда использовать

- Проектирование БД.
- Создание ERD.
- Формирование словаря данных.
- Event Storming.

## Выходные артефакты

- Список сущностей.
- Атрибуты сущностей.
- Связи между сущностями.
- Кандидаты в агрегаты.
