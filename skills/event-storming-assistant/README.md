# Event Storming Assistant

## Назначение

Навык предназначен для выявления команд, событий, агрегатов и политик на основе бизнес-процесса.

## Когда использовать

- Проектирование микросервисной архитектуры.
- Проектирование Kafka-интеграций.
- Определение границ сервисов.
- Event Driven Architecture.
