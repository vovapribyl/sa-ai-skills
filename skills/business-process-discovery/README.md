# Business Process Discovery

## Назначение

Навык предназначен для выделения бизнес-процессов на основе бизнес-требований или расшифровки интервью с заказчиком.

Навык помогает подготовить материал для последующего построения BPMN-диаграмм.

---

## Когда использовать

Используйте навык после формирования бизнес-требований.

Навык позволяет определить:

- бизнес-процессы;
- участников процессов;
- точки принятия решений;
- основные сценарии;
- альтернативные сценарии;
- входные и выходные данные процессов.

---

## Входные данные

- Business Requirements Document (предпочтительно);
или
- расшифровка встречи;
или
- описание бизнес-процесса.

---

## Выходные артефакты

- Каталог бизнес-процессов.
- Основной сценарий процесса.
- Альтернативные сценарии.
- Участники процесса.
- Точки принятия решений.
- Точки интеграции.
- Список вопросов для уточнения.

---

## Ограничения

Навык не строит BPMN автоматически.

Его задача — подготовить структурированное описание процессов, достаточное для последующего моделирования.
